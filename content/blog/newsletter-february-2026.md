**🚀 Deployments Now Default to Infinite Strategy**

We’ve upgraded deployment reliability across the board.

All deployments now default to **infinite restart strategy**.

What this means:

- If a job crashes → it restarts automatically
- If a node fails → it recovers
- Your service stays online without manual configuration

No extra setup. No complex config. Just more uptime by default.

**[Try it now](https://deploy.nosana.com)**

**🔌 Nosana GPUs Are Now Live on OpenGPU**

Nosana GPU supply is now integrated into OpenGPU’s free-market network.

With this integration:

- OpenGPU can route AI workloads to Nosana
- Execution happens through OpenGPU’s Relay layer
- More real GPU supply enters the open market

This strengthens distributed compute infrastructure while giving builders more execution flexibility.

**[Read the full blog](https://nosana.com/blog/nosana_opengpu/)**

**🤝 Nosana × Sallar: Distributed Compute at Massive Scale**

We’re partnering with Sallar to expand distributed compute access.

Here’s how the collaboration works:

- **Sallar** brings a massive global network of smartphones contributing compute
- **Nosana** provides high-performance GPU Cloud capacity
- Together, we cover both lightweight distributed workloads and heavy GPU-intensive AI jobs

This combination creates a more complete distributed compute stack for builders.

**[Learn more](https://nosana.com/blog/nosana_partners_with_sallar/)**

**🛡 Nosana Partners with Alio to Power Agentic AI Security**

Agentic AI requires serious security infrastructure.

Alio now uses Nosana GPU Cloud to:

- Run large-scale LLM risk detection
- Execute red-teaming simulations
- Avoid high centralized cloud costs
- Eliminate long-term lock-in risks

This partnership shows how decentralized GPU Cloud can power advanced AI security workloads at scale.
